# Python货物进销管理系统 python763

## 项目概述

本项目是一个基于Python Flask框架的货物进销管理系统。它涵盖了登录注册、采购订单管理、商品信息管理、供应商管理、销售管理、仓库管理以及成员管理等功能模块。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Flask
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 运行环境

- Python 3.0及以上
- MySQL 8.0

## 功能模块

- 登录注册
- 采购订单管理
- 商品信息管理
- 供应商管理
- 销售管理
- 仓库管理
- 成员管理
- 个人信息修改

## 系统角色

- 系统管理员
- 采购人员
- 销售人员
- 仓库管理员
- 普通用户

## 目录结构

```
项目目录/
│
├── app/               # Flask应用目录
│   ├── static/        # 静态文件目录
│   ├── templates/     # HTML模板目录
│   └── views/         # 视图层目录
│
├── config.py          # 配置文件
├── models.py          # 数据模型
├── run.py             # 应用启动文件
│
└── venv/              # 虚拟环境目录（含依赖）
```

## 部署步骤

1. 确保Python 3.0以上版本以及MySQL 8.0数据库环境已经安装。
2. 克隆或下载项目代码至本地。
3. 在项目目录下建立虚拟环境并激活。
4. 安装项目依赖。
5. 配置数据库连接信息。
6. 运行`run.py`启动应用。

- 功能完善，涵盖了进销存管理的主要流程。
